Commands:
  1. Know what directory you are in, use command: pwd
  2. Create a new directory, use command: mkdir "name of new directory" Example: mkdir git-new
  3. Enter a directory, use command: cd "name of directory" Example: cd git-new
  4. Create files in a directory, use command: touch "filename" Example: touch README.md
  5. To create and replace text in a file, use command: echo "text" > README.md
  6. To create and add text in a file, use command: echo "text" >> README.md
  7. Copy file, use command: cp "file to be copied" "where to be copied" Example cp README.md git-more
  8. Delete file, use command: rm "filename" Example: rm README.md
  9. Initialize git repositoy, use command: git init
  10. Stage change to a file, use command: git add .
  11. Commit staged changed, use command: git commit -m "text"
  12. Check the status of your files, use command: git status
  13. Create and switch to a new branch, use command: git checkout -b branchname Example: git checkout -b feature-branch
  14. Switch to any branch, use command: git switch branchname Example: git switch master
  15. Verify where the HEAD is pointing to, use command: git log
